Unimportant and Urgent
  1. Task 1: Save the quadrants
    Pressing Crtl or Cmd + Shift + S or going to File > Save Quadrant will save the quadrants into a .qdr file of your choice which you can place anywhere you like on your device. This allows you freedom to have and manage different QDRs for different purposes at the same time. Tip: Saving the quadrants with meaningful namesmakes it easier for referencing.
    Time of task creation shows here
  2. Task 2: Loading the quadrants
    Pressing Ctrl or Cmd + Shift + O or going to File > Load Quadrant will open your qdr file.

Important and Not Urgent
  1. Task 1: Opening the settings
    Press Ctrl or Cmd + K to open the settings page. You can change the name of the quadrants from the settings page, you can also change the number of days you want to wait before you clear the complete list.
    Time of task creation shows here
  2. Task 2: Restoring defaults
    Don't worry if you mess things up or if you just want to go back to a fresh start. You can restore the default settings. Just go to Actions > Settings > Restore defaults.
